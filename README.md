# 🔐 Secure File Storage System with AES

A premium cybersecurity desktop application built using Python and Tkinter for secure AES-256 file encryption and decryption.

✅ Secure File Decryption

# ⚙️ Installation

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Generate AES Key

```bash
python generate_key.py
```

## Run Application

```bash
python main.py
```
